An Amazing Slack App That Helps Team Create Polls.

Slagora is one of those ready made slack applications that enable teams to create interactive polls using preferential voting algorithms.

Scroll Down

Easy to use slack poll app.

Slagora makes it easy for slack teams to create polls that will enable the team take informed decisions. It is easy to setup and use and can be maid available for every user in the team.

Light Weight.

Slagora make use of the Agora vote counting library which has proven to be one of the fastest and accurate libraries used to count votes in a poll or election.

User-Friendly.

The UI of Slagora is simple and can easily be understood by any Slack user. It exposes a range of slash command and a Bot that can be used to interact with the application

Powerful.

Slagora can handle a huge number of polls/elections and readily provide results when necessary. It is built on top of the Play framework and uses the Agora library as vote counting library, accounting to it's efficient nature.

Free and Open Source.

Slagora is developed by the Australian Open Source Software Innovation and Education (AOSSIE) team. It is free and open source, follow us on Gitlab. Love our work? Star the project on Gitlab

How The App Works?

Sign Up

For Slack users to make use of what Slagora has to offer. Someone has to install the application in a channel and users can start making use of the application.

Create

User can then create polls/elections using slash commands. Polls/Elections can only be created in channels and groups by registered users.

Vote

Upon creating the poll/election Slagora will send a message to the said channel inviting other members to vote. Only registered users can vote, however everyone could be able to see the results of the poll/election

Results

When a poll/election is due, the results of the poll/election will be sent to the channel. However if the poll/election has as option real time results, users can request for the results at anytime.

Loaded With Features You Would Surely Love.

Slagora come with a good number of features which enable users to make the most out of the application.

Cloud Based

Slagora is cloud base and hosted on Heroku and uses Mongo DB as backing store. Polls/Elections created with Slagora can run for as long as the application is live.

Always Secure

Data stored in Slagora is not shared with any third party and is not shared across different teams in slack even if the a user is a member of the other teams.

Bot Chat

Coming Soon. User will be able to interact directly with Slagora Bot. This will enable the Bot to help users create, edit, delete polls/elections.

Free

Slagora is free and open source. Teams can easily install the app with no payment required. Contributors are welcome to send us feedback and help us improve on the application
